## Overview

这是一个为理工科学生设计的个人网站，整合个人介绍、项目、技术笔记、科研兴趣和在线简历。

## Motivation

个人网站应该既能展示成果，也能沉淀学习过程。使用静态站点和 Markdown 可以降低长期维护成本。

## Methods

- 使用 Astro 静态生成页面
- 使用 Content Collections 管理 blog 和 projects
- 使用统一组件保持视觉一致
- 使用简洁 CSS 保持低复杂度

## Tech Stack

Astro, TypeScript, Markdown, CSS.

## Results

占位：补充 Lighthouse 分数、部署链接和未来迭代计划。

## My Contribution

完成信息架构、视觉系统、内容模型、页面实现和部署文档。

## Challenges

需要在学术感、工程感和个人表达之间保持平衡。

## What I Learned

内容模型比炫酷效果更重要，长期可维护的网站应该让更新内容变得非常轻。
